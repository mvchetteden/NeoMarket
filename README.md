# NeoMarket

NeoMarket — это учебный макет маркетплейса на Vue 3 и TailwindCSS. Репозиторий содержит статический HTML, подключающий библиотеки через CDN.

## Запуск

Для просмотра достаточно открыть файл `index.html` в современном браузере. Интернет‑соединение требуется для загрузки Vue, TailwindCSS и FontAwesome с CDN.

## Структура

- `index.html` — основная страница приложения.
- `assets/styles.css` — таблица стилей.
- `assets/app.js` — скрипт Vue‑приложения.
